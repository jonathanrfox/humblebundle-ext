<h1>Directions</h1>
<code>
$ git clone https://github.com/jrusty/humbundlr.git
$ cd humbundlr
$ npm run build:prod
$ npx web-ext run --firefox /path/to/firefox --source-dir ./dist
</code>

<h1>Requires</h1>
<ul>
  <li>Firefox > 59</li>
</ul>
